# Breadcrumb 面包屑

显示当前页面的路径，快速返回之前的任意页面。

## 基本用法

适用广泛的基础用法。

<fox-breadcrumb>
<fox-breadcrumb-item to="name">首页</fox-breadcrumb-item>
<fox-breadcrumb-item to="age">组件</fox-breadcrumb-item>
<fox-breadcrumb-item>面包屑</fox-breadcrumb-item>
</fox-breadcrumb>

## 自定义分隔符

<fox-breadcrumb separator="/">
<fox-breadcrumb-item to="name">首页</fox-breadcrumb-item>
<fox-breadcrumb-item to="age">组件</fox-breadcrumb-item>
<fox-breadcrumb-item>面包屑</fox-breadcrumb-item>
</fox-breadcrumb>
